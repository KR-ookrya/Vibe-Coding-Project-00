01 — 문제 정의 & 비전

왜 이 서비스를 만드는가

취준생이 겪는 문제
01

전문 첨삭 비용이 1건당 3만~10만 원 — 반복 첨삭은 부담이 크다.

02

주변에 솔직한 피드백을 줄 현직자·경력자를 찾기 어렵다.

03

혼자 수십 번 고쳐도 어디가 문제인지 객관적으로 알기 힘들다.

→ Claude Opus 4.7의 언어 이해 능력으로 10년 경력 컨설턴트 수준의 심층 피드백을 즉시·무료·무제한으로 제공한다.
핵심 가치

즉각성

입력 즉시 스트리밍으로 결과를 실시간 확인

구체성

점수·근거·수정 예시까지 포함한 실질 피드백

접근성

브라우저만 있으면 — 회원가입·설치 불필요

신뢰성

직무·회사 맥락을 반영한 맞춤 분석

02 — 설계 & 아키텍처

시스템 아키텍처

ADR-0001 · 프론트엔드

Vanilla HTML / CSS / JS

선택 → 빌드 없이 즉시 서빙, fetch Stream Reader로 스트리밍 수신

ADR-0002 · 상태 관리

Vanilla JS · DOM 직접 제어

선택 → 상태가 단순해 Redux 등 라이브러리는 과한 설정 비용

ADR-0003 · 백엔드 & AI

Express + Claude Opus + SSE

선택 → 긴 응답에 SSE 필수, Prompt Caching으로 반복 비용 절감
